Installation Guide

# Stable version
pip install coresec

# Development version
git clone https://github.com/CoreSecFrame/CoreSecFrame.git
cd CoreSecFrame
pip install -e .

Quick Start

# Initialize project
coresec init my-project

# Run network scan
coresec scan network --target 192.168.1.0/24

# Generate report
coresec report generate --format html

CLI Reference

Network Scanning

coresec scan network [options]

Options:

  • --target: IP range or host
  • --profile: Scan intensity (light|medium|full)
