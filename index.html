Open-Source Python Framework for AIO Cybersecurity Operations

Secure by Design, Powerful by Default

CoreSecurityFramework provides a modular architecture for building custom security tools and automating penetration testing workflows.

# Install locally
git clone https://github.com/CoreSecFrame/CoreSecFrame.git
cd CoreSecFrame
pip install -r requirements.txt
python setup.py
